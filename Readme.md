# PySide2-UI Converter

## Usage

1. In the first field, input the .ui file that you want to convert to a .py file
2. In the second file, input the final name and path that you want the converted .py file to be.
3. If you want to save this setup, go to File / Save Template to save the setup out. You can call these settings back using the dropdown menu on the top right of the screen.
